# StartupLens API Backend

This is the backend API handling all of the AI logic (Groq / Google Gemini / RAG) to process ideas, market sizes, and competitors. It expects variables to be passed securely to the environment (e.g., `GOOGLE_API_KEY`, `GROQ_API_KEY`).

Deployed directly from GitHub to Hugging Face Spaces.
